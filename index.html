HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • <!DOCTYPE html> is a declaration that tells the browser what document type it shoudl expect when it is opened.
  • Search engine optimization (SEO) is the process of improving the ranking of a website in serach engines.
  • The style element defines the CSS styles associate with a website
  • The base element defines the base URL for a webpapge
  • The link element connects an external resources to the HTML document
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • The charset element sets our character encoding.
  • The http-equiv element sets an HTTP header for content.
  • The viewport element sets the default view for our webpage.
  • Elements nested inside other elements are call child elements.
  • The element that holds all the children is call the parent element.
  • Element and tag are often used interchangeably. Element is entire element from opening and closing tags. A tag is only what appears inside angle brackets.
  • src defines source of image file
  • alt contains text that describes appearance and functionaily of the image, and can improve search engine rankings

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • 3 ways to style webpage with CSS: inline CSS, internal CSS style sheet, and external CSS style sheet (most common).
  • Declarations contain 2 components, CSS property and value of property
  • *, A special wild card symbol is used to apply a rule to all elements visible on the page.
  • rel attribute specifies the relationship bewtween the current document and the linked document resource
  • .card: The period preceding card designates this as a class selector. Any HTML element assigned to the class card, will gain these property assignments from the CSS
  • Assigning colors in CSS can be done semantically and by its hexadecimal code

Git

  • git status: checks what branch we are currently on
  • git checkout branch-name: navigates to branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd navigates to a subfolder.
  • code . opens VS code.
  • git add -A: commit all files in the working branch
  • git commit -m message: associate a message with the commit. should be a short descriptive message of the changes.
  • git pull origin main: pull in the base branch. good habit to pull base branch before pushing up any code
  • git push origin branch-name: pushes up all of our changes
  • touch file-name will create a file

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • the .js file extension indicates that this file is a JavaScript file.
  • use var to declare a variable
  • console refers to a test environment that developers use to check out their code
  • .log() is a method which means a set of instructions that can be executed by a computer
  • when a computer encounters if statement in a JavaScript file, it will check to see if the condition is truthy
  • falsy values: 0, -0, On, "", null, undefined, NaN, or a Boolean value of false
  • ==== is a strict equality operator
  • a for loop uses the predictable pattern of indices to perform a task on all the items in an array
  • a group of data in a single variable can be stored as an array
  • a string needs to be contained in quotation marks
  • a for loop contains 3 important statements: the starting point for our loop, condition, and the thing that allows the array to iterate, i.e. x++ will increase by 1 for each loop
  • a function is simply a set of instructions that tells the computer how to perform a certain task
  • a function's name is followed by parentheses
  • use curly brackets {} to contain the code for the task we want the function to perform
  • "Math.floor()" and "Math.randon()" are both properties of JavaScript's built-in math object that can be used to generate a random number
